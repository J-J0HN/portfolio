RECENT PROJECTS

California Housing Price Prediction Machine-Learning Model

Identified, implemented, and optimized machine learning algorithms to exhibit high predictive accuracy in forecasting median house values within California location. Used Python Notebook, pandas, numpy, tensorflow, keras, matplotlib and seaborn. Dataset from Kaggle.

Object Detection Speed Camera Program

Python-based program that can live detect movement using computer vision techniques including YOLOv4. Detects vehicles in uploaded videos exceeding a specified speed limit. Saves the video segments that capture the detected motion with annotated information to help users review the incidents.

Companies House Application

Work-in-progress command-line application that allows users with their own Companies House API key to gather various data of any company in the UK. Data fetched include title, address, number, status, creation, type, and address. Written in Rust for faster compilation.
